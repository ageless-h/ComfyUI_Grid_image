# ComfyUI_Grid_image

ComfyUI 节点 - 将多张图像拼接成网格图

## 功能特性

- **动态图像输入**: 连接一个图像后自动出现下一个，最多支持32张图像
- **预设分辨率**: 包含4K、2K、1080p、720p、480p等常用分辨率
- **横竖屏支持**: 16:9、21:9、9:16等常见比例
- **网格布局**: 可灵活设置行数和列数（1-8行/列）
- **顺序编号**: 每个网格右下角自动显示顺序号

## 安装方法

### 手动安装

1. 打开 ComfyUI 的 `custom_nodes` 目录
2. 克隆本仓库:
   ```bash
   git clone https://github.com/yourusername/ComfyUI_Grid_image.git
   ```
3. 重启 ComfyUI

### 通过 ComfyUI Manager 安装（推荐）

1. 打开 ComfyUI Manager
2. 搜索 "ComfyUI_Grid_image"
3. 点击安装

## 使用方法

1. 在 ComfyUI 中搜索 "网格图像" 节点
2. 选择输出分辨率（支持多种预设）
3. 设置网格的行数和列数
4. 连接图像（自动动态扩展，最多32张）
5. 运行工作流，输出拼接后的网格图

## 输入参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| resolution | Combo | 1080p (1920x1080) 16:9 | 输出图像分辨率 |
| rows | Int | 2 | 网格行数 (1-8) |
| cols | Int | 2 | 网格列数 (1-8) |
| images | Autogrow | - | 动态图像输入 (1-32张) |

## 支持的分辨率

### 横屏 16:9
- 4K (3840x2160)
- 2K (2560x1440)
- 1080p (1920x1080)
- 720p (1280x720)
- 480p (854x480)

### 横屏 21:9
- 1080p (1920x810)

### 竖屏 9:16
- 4K (2160x3840)
- 2K (1440x2560)
- 1080p (1080x1920)
- 720p (720x1280)
- 480p (480x854)

## 输出

- `IMAGE`: 拼接后的网格图像
